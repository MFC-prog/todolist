# 📝 To-Do List App

Une application simple de To-Do List développée avec **HTML**, **CSS** et **JavaScript**.

 Fonctionnalités

- Ajouter une tâche
- Supprimer une tâche
- Cocher une tâche comme terminée
- Design responsive


Technologies utilisées

- HTML5
- CSS3
- JavaScript
